# 회원가입 프로그램

## 프로그램 소개

Java를 활용한 콘솔 기반 회원가입 프로그램이다.

사용자로부터 아이디, 비밀번호, 이름, 생년월일, 이메일 정보를 입력받아 회원가입을 진행한다.

회원 정보는 ArrayList와 HashMap을 사용하여 저장하였다.

---

## 개발 환경

- Java
- IntelliJ IDEA

---

## 주요 기능

### 회원가입 진행 여부 선택

- y 입력 시 회원가입 진행
- n 입력 시 프로그램 종료
- 잘못된 입력 시 재입력 요청

### 비밀번호 확인 기능

비밀번호와 비밀번호 확인 값이 일치하는지 검사한다.

### 생년월일 입력 검증

생년월일 입력 시 6자리인지 확인한다.

### 회원 정보 저장

입력받은 회원 정보를 HashMap에 저장하고,
ArrayList에 추가하여 여러 회원 정보를 관리할 수 있도록 구현하였다.

---

## 사용 기술
